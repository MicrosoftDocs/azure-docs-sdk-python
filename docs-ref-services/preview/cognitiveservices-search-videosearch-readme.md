# Microsoft Azure Cognitive Services Video Search SDK for Python

This is the Microsoft Azure Cognitive Services Video Search Client Library.
This package has been tested with Python 2.7, 3.6+.
For a more complete view of Azure libraries, see the [azure sdk python release](https://aka.ms/azsdk/python/all).


## Usage




For code examples, see [Cognitive Services Video Search](https://docs.microsoft.com/python/api/overview/azure/cognitive-services) on docs.microsoft.com.
